Lab 8: User interface for the intersection of a route with a given polygon and calculation of distances


The following list shows the distances (in km) resulting from the intersection between a given route and a given polygon. The list is sorted in an ascending order.

{{ routeCutByPolygon(Route, Polygon) }}

Furthermore the sum of all distances (in km) is given:

{{ sumIntersections(Route, Polygon) }}

To calculate the distances resulting from the intersection with the given polygon for any route please enter a route in the following text field.
Please note: The route must be given as GeoJSON feature collection of type Line String. (To create a GeoJSON please see: https://geojson.io/)


The following list shows the distances (in km) resulting from the intersection between the submitted route and the given polygon. The list is sorted in an ascending order.

{{UserInput_ListDistances}}

Furthermore the sum of all distances (in km) is given:

{{UserInput_SumDistances}}

The given polygon as GeoJSON is:

{{PolygonGeoJSONString}}
